# Engineering

Skills I use daily for code work.

- **[ask-matt](./ask-matt/SKILL.md)** - Router for the Matt Pocock skill set.
- **[diagnosing-bugs](./diagnosing-bugs/SKILL.md)** - Current upstream diagnosis loop for hard bugs and performance regressions.
- **[diagnose](./diagnose/SKILL.md)** - Compatibility alias for `diagnosing-bugs`.
- **[codebase-design](./codebase-design/SKILL.md)** - Shared vocabulary for deep modules, small interfaces, seams, and adapters.
- **[domain-modeling](./domain-modeling/SKILL.md)** - Build and sharpen project domain language through `CONTEXT.md` and ADRs.
- **[grill-with-docs](./grill-with-docs/SKILL.md)** - Grilling session that challenges your plan against the existing domain model and updates docs inline.
- **[triage](./triage/SKILL.md)** - Triage issues through a state machine of triage roles.
- **[improve-codebase-architecture](./improve-codebase-architecture/SKILL.md)** - Find deepening opportunities in a codebase, informed by domain language and ADRs.
- **[implement](./implement/SKILL.md)** - Implementation loop for issue-backed work.
- **[setup-matt-pocock-skills](./setup-matt-pocock-skills/SKILL.md)** - Scaffold the per-repo config that the other engineering skills consume.
- **[tdd](./tdd/SKILL.md)** - Test-driven development with a red-green-refactor loop.
- **[resolving-merge-conflicts](./resolving-merge-conflicts/SKILL.md)** - Resolve in-progress merge or rebase conflicts.
- **[to-issues](./to-issues/SKILL.md)** - Break a plan, spec, or PRD into independently-grabbable issues.
- **[to-prd](./to-prd/SKILL.md)** - Turn the current conversation context into a PRD.
- **[prototype](./prototype/SKILL.md)** - Build a throwaway prototype to flesh out a design.
- **[review](./review/SKILL.md)** - Upstream in-progress review skill.
- **[grill-me](./grill-me/SKILL.md)** - Stateless grilling flow for plans outside a codebase.
- **[grilling](./grilling/SKILL.md)** - Reusable grilling loop used by other skills.
- **[handoff](./handoff/SKILL.md)** - Preserve a thread into a handoff file for a fresh session.
- **[teach](./teach/SKILL.md)** - Stateful learning flow.
- **[writing-great-skills](./writing-great-skills/SKILL.md)** - Reference for writing and editing skills.
- **[zoom-out](./zoom-out/SKILL.md)** - Local legacy skill retained even though upstream v1.0.0 removed it.
